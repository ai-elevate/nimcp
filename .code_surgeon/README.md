# Code Surgeon GPT-5 Integration

Code Surgeon now uses GPT-5 with comprehensive NIMCP project context for intelligent auto-fixing of test failures.

## Configuration
- LLM Provider: OpenAI GPT-5
- API Key: Loaded from environment (OPENAI_API_KEY)
- Context: Full project architecture, patterns, and common failure modes

## Features
- Automatic test failure diagnosis
- Context-aware fix generation
- Minimal, surgical code changes
- Pre-commit hook integration
