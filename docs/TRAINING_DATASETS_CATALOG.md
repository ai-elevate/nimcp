## 🔐 EPISTEMIC FILTERING NOTES

All training data will pass through NIMCP's epistemic filtering system:

### Automatic Adjustments:
- **Facts** (e.g., "Water is H₂O") → Full confidence
- **Opinions** (e.g., "Art X is beautiful") → Reduced confidence (50-80%)
- **Historical claims** → Context-dependent (requires sources)
- **Religious/mythological** → Cultural knowledge, not factual assertions
- **Political statements** → Detected as opinion, multiple perspectives preserved
- **Conspiracy patterns** → Severely penalized (20% confidence)

### Bias Detection Active:
- Confirmation bias
- Bandwagon effect
- Authority bias
- Motivated reasoning
- 12 total bias types monitored

### Sagan Standard Applied:
"Extraordinary claims require extraordinary evidence"
- Novel scientific claims need multiple sources
- Historical revisionism flagged for verification
- Unverified claims learned weakly
