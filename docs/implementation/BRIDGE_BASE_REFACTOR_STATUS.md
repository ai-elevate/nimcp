# Bridge Base Refactoring Status

## Overview
Refactoring ALL bridge files in `/home/bbrelin/nimcp/src/glial/` to use the bridge_base OO pattern.

## Pattern Requirements
1. Add `bridge_base_t base;` as FIRST member of bridge struct in header
2. Replace ALL pthread_mutex_* calls with BRIDGE_LOCK/BRIDGE_UNLOCK macros
3. Use BRIDGE_CREATE_BEGIN and BRIDGE_DESTROY macros
4. Create accessor macros for system pointers (e.g., `#define FOO_GET_SYSTEM_A(bridge) ((type*)(bridge)->base.system_a)`)
5. Use `bridge_base_connect_a/b()` for connections (outside lock!)
6. Use `bridge_base_record_update()` in update functions (inside lock)
7. Replace bio-async functions with `BRIDGE_DEFINE_BIO_ASYNC_FUNCS_TYPE(prefix, type)` macro

## Standard Refactoring Template

### Header File Changes

```c
// 1. Add bridge_base.h include
#include "utils/bridge/nimcp_bridge_base.h"

// 2. Update struct definition
typedef struct {
    bridge_base_t base;  // MUST BE FIRST

    // Keep all other fields EXCEPT:
    // - Remove: void* mutex (or nimcp_mutex_t* mutex)
    // - Remove: bio_module_context_t bio_ctx
    // - Remove: bool bio_async_enabled
    // - Remove: uint64_t last_update_time
    // - Remove: uint64_t total_updates

    // All other fields stay as-is
} foo_bridge_t;

// 3. Add accessor macros after struct definition
#define FOO_GET_SYSTEM_A(bridge) ((system_a_type*)(bridge)->base.system_a)
#define FOO_GET_SYSTEM_B(bridge) ((system_b_type*)(bridge)->base.system_b)
```

### Implementation File Changes

```c
// 1. Update create function
foo_bridge_t* foo_create(config, system_a, system_b) {
    // Validate inputs
    if (!system_a) return NULL;
    if (!system_b) return NULL;

    // Use BRIDGE_CREATE_BEGIN macro
    BRIDGE_CREATE_BEGIN(foo_bridge_t, bridge, BIO_MODULE_FOO, "foo_bridge");

    // Apply config...

    // Connect systems OUTSIDE any locks
    bridge_base_connect_a(&bridge->base, system_a);
    bridge_base_connect_b(&bridge->base, system_b);

    // Initialize other fields...

    return bridge;
}

// 2. Update destroy function
void foo_destroy(foo_bridge_t* bridge) {
    BRIDGE_DESTROY(bridge);
}

// 3. Add bio-async macro (replaces all bio-async functions)
BRIDGE_DEFINE_BIO_ASYNC_FUNCS_TYPE(foo, foo_bridge_t)

// 4. Update all functions using mutex
int foo_update(foo_bridge_t* bridge) {
    BRIDGE_NULL_CHECK(bridge);

    // Get systems via accessors
    system_a_type* sys_a = FOO_GET_SYSTEM_A(bridge);
    system_b_type* sys_b = FOO_GET_SYSTEM_B(bridge);

    BRIDGE_LOCK(bridge);

    // ... update logic ...

    // Record update in base
    bridge_base_record_update(&bridge->base);

    BRIDGE_UNLOCK(bridge);

    return 0;
}

// 5. Replace all bridge->mutex with BRIDGE_LOCK/UNLOCK
// 6. Replace all bridge->system_a/b with accessor macros
// 7. Replace bridge->bio_async_enabled with bridge_base_is_bio_async_connected(&bridge->base)
// 8. Remove manual bio-async connect/disconnect/is_connected functions
```
