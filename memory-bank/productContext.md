# Product Context

## Why This Project Exists
Current AI trends favor large language models with exponentially growing resource requirements:
- **Power/water consumption** growing unsustainably
- **Hallucinations** remain unsolved due to stochastic nature
- **AGI uncertainty** — unclear if transformers can achieve general intelligence

NIMCP provides an alternative: biologically-grounded neural computing that models how evolution already solved intelligence.

## Problems It Solves
1. **Resource efficiency** — runs on IoT to supercomputers, not just data centers
2. **Hallucination-free** — grounded in biological mechanisms, not statistical patterns
3. **Continuous learning** — real-time adaptation, not frozen post-training
4. **Ethical by design** — moral reasoning built into architecture
5. **Hardware flexibility** — CPU, GPU, neuromorphic, quantum, FPGA support

## How It Should Work
- **Brain simulation**: Faithful modeling of neurons, synapses, brain regions, glial cells
- **Distributed cognition**: Swarm intelligence across multiple nodes
- **Adaptive tiers**: Portia system adjusts fidelity based on available compute
- **Self-monitoring**: Immune system detects anomalies, introspection enables self-improvement
- **Sleep consolidation**: Memory replay and synaptic homeostasis during rest cycles

## User Experience Goals
Developers integrating NIMCP should:
- Access clean C API for brain operations
- Scale from embedded devices to server clusters
- Leverage GPU/neuromorphic acceleration transparently
- Monitor brain health through immune system metrics
- Configure ethical constraints for their use case

## Biological Systems Modeled
| System | Biological Basis | NIMCP Implementation |
|--------|------------------|---------------------|
| Neurons | Action potentials, synaptic transmission | Spiking networks, LTC dynamics |
| Brain Regions | Cortical columns, hippocampus, PFC | Modular brain architecture |
| Plasticity | LTP/LTD, STDP, metaplasticity | Multiple plasticity mechanisms |
| Neuromodulation | Dopamine, serotonin, ACh, norepinephrine | Spatial diffusion model |
| Glial Cells | Astrocytes, oligodendrocytes, microglia | Full glial integration |
| Immune System | B cells, T cells, cytokines | Brain immune with BBB |
| Sleep/Wake | Circadian rhythms, consolidation | Sleep cycles with replay |
| Hemispheric | Left/right brain, corpus callosum | Bilateral processing |
| Swarm | Ant colonies, bee hives | Distributed cognition |
