# Project Brief

## Overview
NIMCP (Neuro Inspired Modular Control Protocol) is a C-based biologically-inspired neural computing platform. It implements a complete brain simulation with 451,000+ lines of code across 2,200+ source files, modeling biological neural systems as an alternative to transformer-based LLMs.

## Core Hypothesis
Evolution has already solved efficient, robust, and general intelligence. NIMCP faithfully models these biological solutions rather than scaling parameter counts.

## Core Requirements
- Run on any computing platform (IoT to supercomputers) via Portia tier system
- Implement biologically-accurate neural mechanisms (spiking networks, plasticity, neuromodulation)
- Support distributed swarm intelligence across multiple nodes
- Maintain ethical foundation with built-in moral reasoning
- Enable continuous real-time learning (not just training-then-inference)

## Goals
- Create environmentally sustainable AI (vs. power-hungry LLMs)
- Eliminate hallucinations through grounded biological computation
- Achieve artificial general intelligence via biological fidelity
- Support neuromorphic hardware for ultra-low power operation
- Enable self-awareness and introspection capabilities

## Target Platforms
| Tier | Platform | Capability |
|------|----------|------------|
| MINIMAL | IoT devices | Reflex responses only |
| BASIC | Smartphones | Pattern matching, basic learning |
| MEDIUM | Drones/Robots | Working memory, attention, planning |
| STANDARD | Laptops | Full regions, learning, swarm-capable |
| FULL | Servers | All regions, full plasticity, distributed |
| QUANTUM | Supercomputers | Quantum coherence, maximum fidelity |
| NEUROMORPHIC | Specialized chips | Native spiking, ultra-low power |

## Design Principles
1. **Biological fidelity** — model real neural mechanisms, not abstractions
2. **Universal scalability** — graceful degradation on constrained hardware
3. **Swarm intelligence** — distributed cognition, emergent collective behavior
4. **Ethical foundation** — primary directive: further human condition
5. **Self-awareness** — IIT 3.0 Phi, Theory of Mind, introspection
