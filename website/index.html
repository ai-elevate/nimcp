Architecture

Six Networks, One Brain

NIMCP doesn't train a single monolithic model. It trains six heterogeneous neural networks simultaneously, with gradient flow across network boundaries.

Adaptive
9-layer diamond, GPU-accelerated backprop
Spiking (SNN)
768 LIF neurons, BPTT surrogate gradients
Liquid (LNN)
Continuous-time ODE, adjoint gradients
Convolutional
4 cortex processors: visual, audio, speech, somato
Fourier (FNO)
Spectral convolution, frequency-domain learning

Energy-conserving, physics-informed dynamics
🧠

Biological Plasticity

Five learning rules at five timescales: STDP (10ms), BCM (50ms), eligibility traces (1s), structural plasticity (10s), homeostatic scaling (60s). Four neuromodulators (dopamine, acetylcholine, norepinephrine, serotonin) modulate learning based on reward and novelty signals derived from training loss.

👶

Developmental Curriculum

Four stages mirroring human cognitive development: sensory awakening, cross-modal naming, feedback learning, abstract reasoning. Each stage builds on representations formed in the previous one. The brain literally grows its cognitive capabilities over time.

🛡

Safety by Design

Ethics module is always created regardless of configuration. LGSS governance evaluates every inference and weight update. Tamper-resistant audit log with CRC32 checksums and monotonic sequence numbers. Safety rules can only get stricter, never looser.

🤖

Edge & Robot Platform

12 sensor types, safety watchdog, motor output. Four drone flight controller interfaces (MAVLink, DJI, Betaflight, Parrot). ROS 2 bridge publishes cmd_vel, subscribes to /imu, /odom, /joint_states. Sim-to-real transfer with domain randomization.

🌐

Swarm Intelligence

Multi-device federation with UDP discovery, Byzantine fault tolerance, and gossip-based gradient aggregation. Each edge brain adapts collective knowledge to its local environment. Theory of Mind emerges from observing other agents' behavior.

💬

Brain-Native Language

Learned vocabulary from neural activation patterns, not an external LLM. Autoregressive decoding with nucleus sampling. Emergent symbols (omega-tokens) represent concepts the brain discovers that don't map to human language. Inner speech loop refines outputs through self-talk iterations.

Get Started

Build in 20 Minutes

NIMCP compiles on any Linux system with a CUDA-capable GPU.

# Clone and build git clone https://github.com/redmage123/nimcp.git cd nimcp/build && cmake .. && make nimcp -j4 # Build Python bindings make nimcp_python -j4 cp build/lib/python/nimcp.so ~/.local/lib/python3.12/site-packages/ # Create a brain and start learning python3 -c " import nimcp brain = nimcp.Brain('my_brain', neuron_count=100000) brain.learn_vector(features, target, label='hello world') "
